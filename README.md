WebOS that was taken over by cats D:

CatOS provides this features:
- top bar with real time
- CatNotes -> app for writing notes about your cats
- Gallery -> app with photos of cats 
- desktop icons that open the apps
- draggable and closable apps

It was created by following tutorial on Hack Club!
It is hosted via GitHub pages function
